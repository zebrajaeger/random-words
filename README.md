# Random Words

Generator for random words as part of a creative process.

![alt text](doc/electron_pjtJmhiPOM.png)

## Develop

- Prerequisites
  - NodeJs >20
  - yarn
- Run the code
  - clone this repo
  - run "yarn install"
  - run "yarn run start"
- Build release
  - run "yarn run package"
- Build setup binary
  - run "yarn run make"
